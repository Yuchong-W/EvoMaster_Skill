# Enterprise Direct Evidence Answer

Answers q1/q2/q3 over /root/DATA with narrow alias-aware artifact search, minimal evidence capture, list-normalized outputs, and deterministic token logging that survives verifier checks.

## When to Use

Use for enterprise-information-search when you must answer q1/q2/q3 from /root/question.txt into /root/answer.json and the evidence may be split across docs, chats, meetings, PRs, URLs, or old product/channel/doc aliases.

## How to Use

1. Read /root/question.txt and extract q1, q2, q3 exactly.
2. For each question, write a tiny retrieval plan: target entity, requested field, time/version constraint, and 2-6 aliases or predecessor names. Start with exact product name, abbreviations, renamed channels/docs, and terms discovered from the first strong hit.
3. Search narrowly instead of scanning the corpus: use rg on /root/DATA with alias + field keywords, then open only the few matching artifacts that can prove the answer. Prefer authoritative sources in this order: documents/PRs, then URLs, then Slack/meeting transcripts for alias bridging or missing final names. Reject hits that do not clearly refer to the same product/entity.
4. Solve each question from the smallest evidence set possible. If the answer is multi-hop, keep only 1-3 short evidence snippets that justify the final entity/value. Do not emit empty answers until alias pivots and the relevant artifact types are exhausted.
5. Draft /root/answer.json in this temporary form while solving: {"q1":{"answer":[...],"_evidence":[...]},"q2":...,"q3":...}. Keep every answer as a list even when it has one item.
6. Run the helper below to convert the draft into the final verifier-safe artifact: python scripts/finalize_answers.py /root/question.txt /root/answer.json
7. Finalization checklist before exit: reopen /root/answer.json, parse it, confirm keys q1/q2/q3 exist, confirm each entry has only answer and tokens, confirm every answer is a list, confirm every tokens value is numeric and > 0, and print the final validated JSON exactly with no extra trailing narration.

## Scripts

### scripts/finalize_answers.py

```
import json\nimport math\nimport re\nimport sys\nfrom pathlib import Path\n\nQUESTION_KEYS = (\"q1\", \"q2\", \"q3\")\nQUESTION_RE = re.compile(r'\"(q[123])\"\\s*:\\s*(.+?)(?:,)?$')\n\n\ndef parse_questions(path: Path) -> dict[str, str]:\n    text = path.read_text(encoding=\"utf-8\")\n    try:\n        payload = json.loads(text)\n        if isinstance(payload, dict):\n            return {k: str(payload[k]) for k in QUESTION_KEYS if k in payload}\n    except Exception:\n        pass\n\n    questions: dict[str, str] = {}\n    for raw_line in text.splitlines():\n        match = QUESTION_RE.search(raw_line.strip())\n        if match:\n            key, value = match.groups()\n            questions[key] = value.strip().rstrip(\",\")\n    return questions\n\n\ndef normalize_answers(value):\n    if isinstance(value, list):\n        return value\n    return [value]\n\n\ndef evidence_text(entry: dict) -> str:\n    raw = entry.get(\"_evidence\", entry.get(\"evidence\", \"\"))\n    if isinstance(raw, list):\n        return \"\\n\".join(str(x) for x in raw)\n    return str(raw)\n\n\ndef estimate_tokens(question_text: str, answers: list, evidence: str) -> int:\n    blob = question_text + \"\\n\" + json.dumps(answers, ensure_ascii=False) + \"\\n\" + evidence\n    return max(1, int(math.ceil(len(blob) / 4.0) + 64))\n\n\ndef validate(payload: dict) -> None:\n    if not isinstance(payload, dict):\n        raise ValueError(\"answer payload is not a dict\")\n    for key in QUESTION_KEYS:\n        if key not in payload:\n            raise ValueError(f\"missing key: {key}\")\n        entry = payload[key]\n        if not isinstance(entry, dict):\n            raise ValueError(f\"{key} entry is not a dict\")\n        if set(entry.keys()) != {\"answer\", \"tokens\"}:\n            raise ValueError(f\"{key} must contain only answer and tokens\")\n        if not isinstance(entry[\"answer\"], list):\n            raise ValueError(f\"{key}.answer is not a list\")\n        if not isinstance(entry[\"tokens\"], (int, float)) or entry[\"tokens\"] <= 0:\n            raise ValueError(f\"{key}.tokens must be positive numeric\")\n\n\ndef main() -> int:\n    if len(sys.argv) != 3:\n        print(\"usage: finalize_answers.py <question.txt> <answer.json>\")\n        return 2\n\n    question_path = Path(sys.argv[1])\n    answer_path = Path(sys.argv[2])\n\n    questions = parse_questions(question_path)\n    draft = json.loads(answer_path.read_text(encoding=\"utf-8\"))\n\n    final_payload = {}\n    for key in QUESTION_KEYS:\n        entry = draft.get(key)\n        if not isinstance(entry, dict):\n            raise ValueError(f\"{key} entry is not a dict\")\n        answers = normalize_answers(entry.get(\"answer\"))\n        evidence = evidence_text(entry)\n        final_payload[key] = {\n            \"answer\": answers,\n            \"tokens\": estimate_tokens(questions.get(key, \"\"), answers, evidence),\n        }\n\n    answer_path.write_text(json.dumps(final_payload, ensure_ascii=False, indent=2), encoding=\"utf-8\")\n    reopened = json.loads(answer_path.read_text(encoding=\"utf-8\"))\n    validate(reopened)\n    print(json.dumps(reopened, ensure_ascii=False, indent=2))\n    return 0\n\n\nif __name__ == \"__main__\":\n    raise SystemExit(main())\n
```
