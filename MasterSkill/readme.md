# EvoMaster_Skill
The project proposes integrating DeepResearch into the skill creation pipeline, leveraging deep web search to supply the model with specialized knowledge it otherwise lacks, thereby narrowing the gap with human-expert skills. Experiments show that even a straightforward integration already yields notable performance gains.
